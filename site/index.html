Private by default

Skyviz is a free, open-source alternative built for collectors who want control. Your file is processed locally in your browser only, and you can choose whether to keep a copy on this device.

Step 1

Start your private dashboard

No upload required. Try a built-in sample deck with 20 popular airports and 20 popular aircraft models (about 575k XP).

Expected after upload: two private local views, Map and Aircraft. Accepted input: Skycards export JSON (for local testing, use skycards_user.json).

Step 2 (optional)

Keep a local copy on this device

Step 3

Need your Skycards export JSON first?

  1. Open the guide: github.com/mfkp/skycards-export
  2. Run one of the official export scripts:
    Windows PowerShell Invoke-WebRequest -Uri https://raw.githubusercontent.com/mfkp/skycards-export/main/skycards_export.ps1 -OutFile skycards_export.ps1 powershell -ExecutionPolicy Bypass -File .\skycards_export.ps1
    macOS / Linux / Git Bash curl -O https://raw.githubusercontent.com/mfkp/skycards-export/main/skycards_export.sh chmod +x skycards_export.sh ./skycards_export.sh
  3. The script saves skycards_user.json in your current folder. Upload that file here.
